# dbBEST (by @sb)

Electron-App mit sqliteDB Anbindung

## Install the dependencies
```bash
npm install
```

### Starte App zum Entwickeln
```bash
npm run dev
```

### Build the app for production
```bash
npm run build
```

### Cypress starten
```bash
npm run cy
```

### JSDOC - Dokumentation erstellen (in ./docs/index.html)
```bash
npm run doc
```
